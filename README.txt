¡Buenas! Para que puedas ejecutar este programa usa el commando "npm i" que instalará todos los módulos que Vue necesitará.

Por problema de Heroku en su versión gratuita, a veces el servidor "se duerme" así que al entrar a veces toca recargar la
página una o más veces para que funcione, también hay un problema con la conexión de Heroku con la API, pues Heroku no
permite conexiones inseguras y la API usa HTTP, por esto toca pedir acceso temporal a un servidor que permite usar la página
sin problema en esta URI: https://cors-anywhere.herokuapp.com/corsdemo

La URL de la app desplegada en Heroku es: https://weather-app-leantech.herokuapp.com
